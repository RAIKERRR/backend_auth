# 🔐 Auth Backend

A backend authentication system built using Node.js, Express, MongoDB, and TypeScript.

---

## 🚀 Features

* User Registration
* User Login
* Password Hashing (bcrypt)
* JWT Authentication (Access + Refresh Tokens)
* Protected Routes
* Input Validation (Zod)

---

## 🛠️ Tech Stack

* Node.js
* Express.js
* MongoDB (Atlas)
* Mongoose
* TypeScript
* JWT (jsonwebtoken)
* bcrypt
* Zod

---
